* 프로그래밍 기법
  * 멀티 프로그래밍 : 한대의 CPU로 여러 개의 프로그램을 동시에 처리
  * 분산처리 프로그래밍 : 지역적으로 분산된 여러 대 컴퓨터 연결, 작업분담처리
  * 일괄처리 프로그매이 : 처리할 데이터를 모았다가 한번에 처리 (Ex: 급여계산)
  * 리얼타임(실시간) : 처리 데이터 발생시 바로 처리  (Ex: 좌석 예약)

* Interrupt cycle에 대한 마이크로 오퍼레이션 중에서 가장 관계가 없는 것은?
